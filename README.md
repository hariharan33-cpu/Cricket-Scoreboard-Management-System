
Cricket Scoreboard Management System
Overview

Cricket Scoreboard Management System is a Java console application that simulates a 10-over cricket match. The project is developed using Object-Oriented Programming (OOP) concepts and dynamically manages players, teams, and match data using Java Collections.

Features
Simulates a complete 10-over cricket match
Dynamic team and player management using ArrayList
Automatic batting and bowling scorecards
Random ball-by-ball match simulation
Match result generation
Console-based user interface
OOP Concepts Used
Abstraction
Inheritance
Polymorphism
Encapsulation
